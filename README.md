## Getting Started

In order to run locally this project type the following commands over the terminal in Linux or Mac or the Windows cmd console:

```s
$ git clone https://github.com/guillainbisimwa/React-Bookstore.git
$ cd React-Bookstore
$ npm install
$ npm start

```
